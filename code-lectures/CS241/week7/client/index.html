Vocabulary

  • Document Object Model - The "Document Object Model" (or DOM) is a representation of the content present on an HTML page. This model represents each element of the page as a node on a "tree" (much like a folder tree in your file explorer.)
  • Promise - A Promise is a special kind of object that captures asynchronous behavior. A promise object has a .then() method that accepts a callback that should run once the asynchronous behavior is complete.

When we make a website, we might start with some static content. However, later on in the page, we might want to create content that responds to an API request or user input. We call this type of content "dynamic content".

In order to create dynamic content, we need to use JavaScript. To use JavaScript in the browser, we add a script tag (typically to the head of the document.) This script tag has an src attribute, just like images and videos. Lets add a script tag that points to a "main.js" file in the "js" folder of this site.

Sometimes, we don't just want to do things immediately when a page loads. Instead, we want to respond to an action a user performs. To do this, we can attach an event to the element where the action occurs. For example, every button in a web page has an "onclick" event. We can access that event in two ways:

  • We can add a function call to the onclick attribute of the button.
  • We can call addEventListener() with the "click" event on the button object in our JavaScript.
