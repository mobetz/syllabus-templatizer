Javascript

JavaScript is a fully-featured programming language. A programming language differs from the markup languages we've used so far in a few important ways. Our markup languages have been 'declarative' -- we write all of the things we want, and the browser figures out how to order and prioritize different rules. By contrast, a programming language is more like giving instructions to a very pedantic person: every instruction will be executed exactly as written, in order.

Today, we will talk about four simple categories of statements we can use in Javascript:

  • Assignment -- the ability to 'save' a piece of data into a 'variable' with the equals sign.
  • Functions -- the ability to 'declare' and later 'call' a whole group of instructions we want to run.
  • Conditional Branches -- the ability to only follow instructions 'if' a condition is met.
  • Loops -- the ability to repeatedly follow instructions 'while' a condition is met.

But first, we need to connect our Javascript code page to our HTML page. This is going to work quite similarly to how we connect CSS to our webpages -- with a tag in the head.
