Course Description

This course introduces the fundamentals of web programming. Students will build database driven Web sites that retrieve, integrate and present database content. Students will use professional tools used to enhance and accelerate the development of web sites. Client-side and server-side scripting will be presented using current web technologies.

Pre-requisite: CS 120 Programming I or CS 118 Scripting.

Required Texts

In lieu of a traditional textbook, this course will make use of Open Educational Resources (OERs). During the first week of class, we will discuss the learning platforms we will be using.

This course provides students with an introduction to programming in a web context. By the end of the course, students are expected to have achieved the following outcomes:
  • Design endpoint-based architectures for executing server-side code.
  • Understand how data is transported between clients and servers.
  • Write server code that interfaces with databases to design data-driven APIs.
    •
  • Achieve fluency in Javascript that works with variables, functions, and conditional logic.
  • Identify common web APIs used for accessing functionality of a browser.
  • Understand the concept of browser sandboxing and how it informs client side program design.
    •
  • Be comfortable with common tools used for web development.
  • Identify evolving web standards for compiling and delivering code that runs in the browser.
  • Understand modern web technologies such as canvas, websockets, and web components.
