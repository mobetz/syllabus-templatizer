Course Description

This course introduces fundamental programming concepts. Students learn to design and implement platform-independent scripts. Students will learn scripting for system-administration, database communication, and to provide user interactivity through the use of multimedia resources. By the end of the course students will have a concrete understanding of programming in several important scripting contexts.

Required Texts

In lieu of a traditional textbook, this course will make use of Open Educational Resources (OERs). During the first week of class, we will discuss the learning platforms we will be using.

This course aims to serve as an introduction to programming with a focus on general computing and the automation of common tasks. By the end of this course, students should understand:
  • Identify statements as the basic building block of scripting.
  • Understand how data is stored and retrieved in a script using variables.
  • Use expressions to modify values in a script.
  • Recognize how if statements allow scripts to conditionally execute instructions.
  • Describe how loops can be used to repeat instructions as long as a condition is met.
  • Explain how functions can be used to create named reusable groups of instructions.
  • Use functions and built-in shell script commands to solve common scripting problems.
  • Understand how to accept input from a user and display output to a user in a script.
  • Write, read, and modify files programmatically in a script.

  • Analyze the difference between textual and numeric representations of data.
  • Contrast loops and branches to understand when the use of each is appropriate.
  • Identify common patterns for writing scripts to validate inputs and sequentially process information.
  • Understand how to break larger problems down into a pipeline of tasks performed on data.
  • Consider common strategies for identifying the source of errors to debug malfunctioning scripts.
